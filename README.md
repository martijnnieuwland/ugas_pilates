# UgasPilates

The public website and CMS for the UgasPilates studio.

## Features

- Content management with TinyMCE editor
- File management with FileBrowser

Note: This is a commercial project for the UgasPilates studio.

## Technologies

- Backend Framework: Django
- Database: PostgreSQL
- Admin Interface: Grappelli
- Rich Text Editor: TinyMCE
- File Management: FileBrowser
- Development Tools: Django Debug Toolbar
